# MesaCards

MesaCards es una PWA móvil de juegos sociales de cartas y fichas virtuales. Está pensada para jugar en celular con pareja o amigos usando puntos, turnos rápidos y una mesa visual elegante.

## Qué incluye esta primera versión

- PWA instalable en celular.
- Diseño mobile-first con modo vertical y adaptación horizontal.
- Banner de instalación para una experiencia tipo app.
- Soporte offline después de cargar una vez.
- Multijugador local/pass-and-play desde el mismo celular.
- Fichas virtuales solo como puntos de diversión.
- 4 juegos base:
  - 21 Flash: reto social contra banca virtual.
  - Hold’em Social: cartas comunitarias y lectura de rivales.
  - Rummy Parejas: robo, descarte, combinaciones y toque.
  - Gem Clash: juego original de gemas, contratos y puntos.

## Próximos pasos recomendados

1. Activar GitHub Pages o desplegar en Vercel/Netlify.
2. Probar desde Android/iPhone.
3. Ajustar sonidos, animaciones y reglas según cómo se sienta jugando.
4. Agregar multijugador online con Supabase Realtime o Firebase.
5. Crear sistema de salas privadas con código.
6. Agregar cuentas opcionales, rankings amistosos y tutoriales guiados.

## Nota de diseño

Las fichas son únicamente puntos virtuales para crear emoción de juego social. No hay premios, pagos ni intercambio de valor.
